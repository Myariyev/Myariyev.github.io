# Mehdi Yariyev - Portfolio Website

This is my personal portfolio website built with Jekyll and hosted on GitHub Pages.

## About

This website showcases my work, projects, and achievements. I am Mehdi Yariyev, a student at ADA University (graduation year 2029), and I have won several robogames festivals.

## Website Structure

- **Home Page**: Introduction and overview
- **About Page**: Detailed information about me, my education, and achievements
- **Projects Page**: Showcase of my projects, including Project 1

## Technologies Used

- Jekyll (static site generator)
- HTML5
- CSS3
- JavaScript
- Font Awesome (for icons)

## Local Development

To run this site locally:

1. Install Jekyll: `gem install bundler jekyll`
2. Install dependencies: `bundle install`
3. Run the server: `bundle exec jekyll serve`
4. Open your browser to `http://localhost:4000`
